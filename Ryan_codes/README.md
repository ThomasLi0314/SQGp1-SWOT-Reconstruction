

# Reconstructing velocity using SQG<sup>+1</sup>

Contain half working code of reconstructing velocity fom SSH using SQG<sup>+1</sup>. The numerics is doing using JAX.

- [Method_summary](https://github.com/Empyreal092/SQGp1_Reconstruction/blob/main/Method_summary.pdf) is the summary of the mathematics and algorithm behind the code. 
- [channel_numbered_notebooks](https://github.com/Empyreal092/SQGp1_Reconstruction/tree/main/channel_numbered_notebooks) contains the notebook used in my [AGU poster](https://github.com/Empyreal092/SQGp1_Reconstruction/blob/main/AGU24_eSQGp1recons_Poster.pdf). These are the version frozen after meeting with Tatsu and Shafer.
- [SQGp1_numbered_notebooks](https://github.com/Empyreal092/SQGp1_Reconstruction/tree/main/SQGp1_numbered_notebooks) contains notebooks to do the same procedure for the data generated by a SQG<sup>+1</sup> model run. It also contains some alternative calculation of $\zeta^1$.
